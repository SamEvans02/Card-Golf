# Card-Golf
The "Golf" card game. <br>
I started making this back in the Spring of 2022 for my CS2 semester project. To my surprise at the time, this was a fun project to work on! It provided me with more unique challenges that I've found only programming could present me with. Making this gave me more of an appreciation for seemingly trivial tasks, such as construction/manipulation of a deck of cards, and how our brains pick up on things so quick and so well. <br>
While the project ended up in a fairly good place when I finished the class, I have always wanted to make it what I know it could be. So, I plan on irregularly continuing my work on this to improve the core mechanics and overall features. <br>
Last, if you haven't ever played this game, give it a shot, it's awesome! <br>
I look forward to seeing what this ends up becoming!
